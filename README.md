Description du projet (FR)

Ce projet consiste à analyser le dataset London Bike Sharing téléchargé sur Kaggle. L’objectif était de comprendre les tendances d’utilisation des vélos à Londres en fonction du temps, de la saison, de la météo et de divers facteurs environnementaux.

Préparation & Transformation des données
• Nettoyage du dataset original
• Renommage des colonnes pour une meilleure lisibilité :
• Saisons de l’année
• Variables météo (température, humidité, vent…)
• Types de météo
• Création d’un fichier Excel enrichi et organisé
• Vérification et conversion des types de données (date, numérique, catégoriel)

Visualisations & Tableau Dashboard

Un dashboard interactif a été créé dans Tableau, incluant :
• Moyenne des vélos utilisés par jour, semaine et mois
• Analyse par saison et type de météo
• Heatmap Température vs Vent pour explorer l’impact climatique
• Évolution temporelle de l’usage des vélos

Ces visualisations permettent d’identifier des schémas d’utilisation et l’influence de la météo sur le partage de vélos.

Compétences acquises
• Nettoyage et préparation de données
• Transformation et renommage de colonnes
• Gestion des types de variables (datetime, numérique, catégoriel)
• Création et manipulation de datasets Excel
• Analyse exploratoire de données (EDA)
• Visualisation avancée avec Tableau (dashboard, heatmap, agrégations)
• Compréhension de l’impact météo/saisons sur l’usage des vélos
• Structuration professionnelle d’un projet data (README, documentation)

Technologies utilisées
• Python (Pandas, éventuellement NumPy)
• Excel
• Tableau
• Kaggle (source des données)

Project Description (EN)

This project analyzes the London Bike Sharing dataset downloaded from Kaggle. The goal was to explore how bike usage in London varies based on time, weather, season, and environmental factors.

Data Cleaning & Transformation
• Cleaned the original dataset
• Renamed columns for clarity:
• Seasons of the year
• Weather variables (temperature, humidity, wind…)
• Weather conditions
• Created an organized and enriched Excel dataset
• Checked and converted variable types (date, numeric, categorical)

Visualizations & Tableau Dashboard

An interactive dashboard was built in Tableau, including:
• Average number of bikes used per day, week, and month
• Seasonal and weather condition analysis
• Temperature vs Wind Heatmap to explore climatic impacts
• Time-series evolution of bike usage

These visualizations highlight patterns and the influence of weather on bike-sharing behavior.

Skills Acquired
• Data cleaning and preprocessing
• Column renaming and structural data improvement
• Handling variable types (datetime, numeric, categorical)
• Dataset building and Excel manipulation
• Exploratory Data Analysis (EDA)
• Advanced Tableau visualization (dashboard, heatmap, aggregations)
• Understanding correlations between weather and bike usage
• Professional project documentation (README, structure)

Technologies Used
• Python (Pandas)
• Excel
• Tableau
